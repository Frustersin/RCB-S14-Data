Bot Command Help (Part 1/2)
Here are all the available commands for this bot:
---------GENERAL COMMANDS---------
!regs
  • Prints out link to regulations
!df
  • Prints out downforce setting for each formula car
!standings <F1><c> <F2><c> Prints out standings from the spreadsheet f.e. !standings F1 for drivers standings f.e. !standings F2C for teams standings !weather
  • Get the current weather conditions for the location.
!rain
  • Shows 5 periods of upcoming rain forecasts at your location.
!lapchecks
  • Roll out lap checks for all positions (1-5) using a custom formula.
  • Displays the chance of being lap-checked, the value rolled, and the status for each position.
---------WEATHER COMMAND---------
!race <league> <round>
  • Retrieve weather conditions at the start of a race.
  • Leagues: f1, f2
  • Rounds: r1, r2, r3, etc.
Examples:
  • !race f1 r1
  • !race f2
    • !race f3 r11
    If no round is specified, it defaults to the nearest upcoming round.
--- !weather
  • Get the current weather conditions.
--- !rain
  • Shows 5 periods of upcoming rain forecasts for the current time.
---
---------RESULTS---------
!results <race>
  • Retrieve the results of a specific race from the spreadsheet.
  • Race: The name of the race or event (e.g., F1_R1, F2_R2, etc.).
Example:
  • !results F1_R1
  • !results F2_R3 The results include the positions, drivers, teams, points, race times, fastest laps, and any penalties.
The driver with the fastest lap will be marked with a star ⭐ next to their time. If the race does not exist or there is an issue retrieving the data, an error message will be displayed.
---------PENALTY POINTS COMMAND---------
!pp [league]
  • Get penalty points for drivers in specified league.
  • Leagues: F1, F2, F3 (optional - shows all if omitted)
Examples:
  • !pp F1 - Shows F1 penalty points
  • !pp - Shows penalty points for all leagues Features:
  • Automatically sorts drivers by points (highest first)
  • Highlights drivers with 12+ points (ban threshold)
---------POLL COMMANDS---------
!poll <duration in hours>, <option1>, <option 2>, ...
  • Create a new poll with specified options and duration.
  • Duration: Time in hours for how long the poll will last.
  • Options: The options users can vote for, separated by commas.
Example:
  • !poll 2, Option 1, Option 2, Option 3
  • Creates a poll for 2 hours with 3 options. The poll will send an anonymous vote button for users to choose an option. The results will be available once the poll ends.
